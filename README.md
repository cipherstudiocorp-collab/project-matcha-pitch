# Project Matcha — Pitch Mockup

Pitch concept site for Project Matcha, a matcha cafe in Portland, Oregon (555 NE Couch St).
Built as a pitch demo by Cipher Studio.

- Static HTML/CSS/JS, no build step
- Deployed via GitHub Pages from `main`
- Hero and ritual sections use compressed MP4 loops (muted, autoplay, loop, playsinline)
